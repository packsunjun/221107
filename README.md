# 221107
## 1.onmouseover 이벤트
* onmouseout 의 반대 개념
* 마우스 포인터가 요소 안으로 들어올 때 + 자식 요소 출입시 발생하는 이벤트
* 요소 영역을 벗어나지 않아도 자식 요소에 들어가거나 나올 때 발생함

## 2.onmouseout 이벤트
* onmouseover 의 반대 개념
* 마우스 포인터가 요소 밖으로 나갈 때 + 자식 요소 출입시 발생하는 이벤트
* 요소 영역을 벗어나지 않아도 자식 요소에 들어가거나 나올 때 발생함
